# Student Features

After logging in, students can:

- View available sports teams.
- View team information and player lists.
- Register or apply for a sports team.
- Update personal profile information.
- Track application or selection status.

---

# Administrator Features

After logging in, administrators can access the Admin Dashboard.

The dashboard may display:

- Total number of students.
- Total number of teams.
- Total number of registered players.
- Team statistics and summaries.

Administrators can also: (optional)

### Team Management

- Create new teams.
- Update team information.
- Delete teams.
- View all teams.

### Player Management

- View registered players.
- Add players to teams.
- Update player information.
- Remove players from teams.

### User Management

- View student accounts.
- Manage user information.
- Enable or disable accounts.

### Reports and Monitoring

- View team rosters.
- Monitor player registrations.
- Generate summary reports.

# Getting Started

1. Open the system in a web browser.
2. Log in using either a Student or Administrator account.
3. Navigate through the available menus.
4. Perform the desired actions based on your account role.
5. Log out after completing your tasks.

---

# Notes

- Usernames and passwords are case-sensitive.
- Only administrators can access the Admin Dashboard and management features.
- Students can only access features assigned to the Student role.
- Change default passwords before deploying the system in a production environment.
